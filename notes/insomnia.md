# Insomnia Notes

**Base URL Environment variable**
- Click `No Environment`
- Click `Manage Environment`
- Add: `"base_url": "http://localhost:3001/api"`
- In HTTP Request URL box, press `CTRL-SPACE` and select `base_url`
